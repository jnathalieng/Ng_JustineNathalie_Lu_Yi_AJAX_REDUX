# Ng_JustineNathalie_Lu_Yi_AJAX_REDUX

This project is a Star Wars character and movie guide created using the Star Wars API (SWAPI).

Users can browse a list of characters and click on a name to view information about the movies they appear in, including the movie title, opening crawl, and poster.

The project was built as part of HW #1 ‚Äì AJAX REDUX and focuses on retrieving and displaying data using AJAX.

## About <a id="about"></a>
This repository contains Justine Ng and Yi Lu‚Äôs submission for the AJAX Redux homework.

The goal of this assignment was to:

Use the Star Wars API (https://swapi.info)
Retrieve data using AJAX
Populate an unordered list with 10 or more characters
Make a second AJAX call when a character is clicked
Display the movie title, opening crawl, and poster image
Include loading and error handling
Make the layout responsive

** This project uses AJAX to retrieve data from the Star Wars API and display character and movie information on the page.
